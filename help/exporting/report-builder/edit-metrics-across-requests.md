# Editar [!UICONTROL métricas] em solicitações no [!UICONTROL Report Builder] {#edit-metrics-across-requests-in-report-builder}

Agora é possível editar [!UICONTROL métricas] em várias solicitações de dados de [!UICONTROL Report Builder] de uma só vez, com opções de adição, substituição e remoção.

>[!VIDEO](https://video.tv.adobe.com/v/23547/?quality=12)

Essa funcionalidade é adicional aos recursos existentes para edição de conjuntos [!UICONTROL de]relatórios, intervalos [!UICONTROL de]datas, [!UICONTROL granularidade]e [!UICONTROL segmentos] entre solicitações. A edição de solicitações em massa pode ser feita no menu de clique com o botão direito do mouse ou em &quot;[!UICONTROL Editar várias]&quot; na janela Gerenciar.

Para obter mais informações sobre esse recurso, visite [a documentação](https://marketing.adobe.com/resources/help/en_US/arb/edit_multiple_metrics.html).
