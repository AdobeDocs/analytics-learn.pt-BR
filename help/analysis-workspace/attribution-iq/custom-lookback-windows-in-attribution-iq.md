# Janelas de pesquisa personalizadas no Attribution IQ

As janelas de pesquisa personalizadas permitem expandir a janela de atribuição além do intervalo de relatórios (até um máximo de 90 dias) e se aplica a cada conversão no intervalo de relatórios. Essa opção normalmente aumentará a precisão da atribuição para eventos que ocorrem no início do período do relatórios, contabilizando as interações que ocorreram no(s) mês(es) anterior(es).
