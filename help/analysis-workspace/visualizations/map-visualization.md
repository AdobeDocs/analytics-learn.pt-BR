#   MapVisualization no Analysis Workspace {#map-visualization-in-analysis-workspace}

Este vídeo mostra como usar o [!UICONTROL Mapa] [!UICONTROL visualização] no Analysis Workspace. 7:00 - Exibição do mapa de calor
