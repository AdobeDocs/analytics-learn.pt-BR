# Baixar 50.000 itens como CSV

A opção Baixar itens como CSV permite baixar até 50.000 itens para uma dimensão em uma tabela de forma livre, com segmentos e filtros aplicados. Embora a interface continue a exibir até 400 itens antes da paginação, essa opção permite acessar mais linhas de dados fora do Analysis Workspace.
